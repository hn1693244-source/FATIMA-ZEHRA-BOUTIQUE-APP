# Automation Scripts

Helpful scripts for development and deployment:

- **setup.sh** - First-time setup (install dependencies)
- **run.sh** - Smart start (auto-detects environment)
- **test.sh** - Run all tests
- **build.sh** - Build services
- **migrate-db.sh** - Run database migrations
- **cleanup.sh** - Clean up resources

Usage: `./scripts/setup.sh`
