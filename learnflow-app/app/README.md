# Application Code

This directory contains the core application:

- **backend/** - 3 FastAPI microservices
  - user-service/ - User authentication & profiles
  - product-service/ - Product catalog & categories
  - order-service/ - Shopping cart & orders
  - chat-service/ - AI chat integration

- **frontend/** - Next.js 16 UI
  - Static export for GitHub Pages
  - Responsive, mobile-first design
  - Tailwind CSS + Shadcn/ui components

- **database/** - Database management
  - migrations/ - SQL migration files
  - seeds/ - Sample data for testing
